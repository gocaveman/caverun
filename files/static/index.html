To ensure things work properly, apply the recommended changes below. Some packages are critical for basic operation (like Go), others are optional and only needed if you use the feature in question (like Docker).
 

Go Version 1.9

Go is installed on your system but is not the latest version. Upgrading is recommended. Download Go

Git Version 2.14.3

Git is installed and is a recent version. No action required. get_app

No GCC Found

GCC is required for CGO to be able to compile C/C++. This is needed for libraries like SQLite which are written in C and called from Go code. Download TDM64-GCC

Docker Not Installed

Docker is required if you wish to run your application in a container. This is recommended for applications that need an external database like MySQL or require other applications in order to work properly. Download Docker

Go Utilities

Some utilities like goimports, stringer, and go-bindata are useful when using go generate. Install Utilities
